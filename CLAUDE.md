# Russian Language Flashcards App

A web application for learning Russian adjective + noun declension (case forms) with Uzbek translations.

## Repository

https://github.com/SardorOga/rus-tili-flashcards

## Project Structure

- `index.html` - Main application (HTML + CSS + JavaScript)
- `data_150.js` - Word data array (`DATA`) containing 150 nouns with adjectives and all case forms
- `sentences.js` - Sentence exercises array (`SENTENCES`) for fill-in-the-blank practice

## Features

1. **Learn (Oʻrganish)** - Browse words with full declension tables
2. **Sentences (Gaplar)** - Fill-in-the-blank exercises with multiple choice
3. **List (Roʻyxat)** - Quick navigation grid of all words

## Russian Cases (Падежи)

1. Именительный (Nominative) - кто? что?
2. Родительный (Genitive) - кого? чего?
3. Дательный (Dative) - кому? чему?
4. Винительный (Accusative) - кого? что?
5. Творительный (Instrumental) - кем? чем?
6. Предложный (Prepositional) - о ком? о чём?
